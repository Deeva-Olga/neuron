Обнаружение объектов на изображении (Для гни)

Описание
Это веб-приложение с поддержкой ONNX Runtime Web для задачи обнаружения объектов на изображении, ориентированное на анализ изображений, связанных с нейронами и глией. Проект позволяет:
1) Загружать изображение.
2) Обнаруживать объекты с помощью модели YOLOv8.
3) Визуализировать результаты обнаружения.
4) Добавлять или удалять объекты вручную.
5) Сохранять отчёт в формате CSV.
6) Вычислять отношение количества нейронов к глии.

Функционал
1) Загрузка изображения
2) Обнаружение объектов
3) Рисование рамок вручную
4) Удаление рамок
5) Скачивание отчёта
6) Расчёт отношения Н/Г
7) Настройка порога вероятности

Технологии
HTML/CSS : Интерфейс и стили.
JavaScript : Логика приложения.
ONNX Runtime Web : Для запуска модели YOLOv8 в браузере.
Canvas API : Отображение изображения и рамок.
File API : Чтение загруженных файлов.

Структура проекта
project/
├── index.html            # Основная страница приложения
├── best.onnx             # Модель YOLOv8 в формате ONNX
└── README.md             # Документация

Как запустить
Убедитесь, что у вас установлен локальный сервер (например live-server, http-server или любой другой).
Поместите файл index.html и best.onnx в одну папку.
Откройте проект через локальный сервер.

Примечания
Приложение оптимально работает с изображениями размером до 1280x1280 пикселей.
Формат модели: YOLOv8s + экспортирована в .onnx.
Поддерживаются классы: 'neuron', 'glia', 'artefacts'.
